THIS TEXT FILE IS INTENDED TO RECORD THE INSTRUCTIONS FOR INTEGRATING THE SCENES INTO THE MAIN FLASH FILE


1. Open main fla
2. On library, create new folder under the "SCENES" folder named <Scene x>
3. From the Scene window, click the "Scene Template" and duplicate it
4. Rename the duplicate scene to <Scene x>
5. Rearrange the <Scene x> to be next after the last scene <Scene x-1>
6. Select scene and go to Timeline. Rename the layers and remove unnecessary layers
7. From the "D:\Short-Animation\Mother's-Message\scenes" folder, go to <Scene x> and open the .fla file for it 
8. Highlight all frames on that <Scene x>.fla, then right click "Copy Frames"
9. Go to Mother's-Message.fla, select the current scene and create new layer and paste all frames there
10. Make sure the timeline elements all stop on the same frame, adjust the other keyframe length if needed esp the timeline default layers like nav buttons
11. Go to actionscript of Scene x then modify the variable names
12. Transfer all loose library elements onto the <Scene x> folder
13. Make sure all Timeline layers are locked for masking to take full effect and set the masker layer to visible
